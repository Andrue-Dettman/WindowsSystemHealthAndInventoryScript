# Windows System Health And Inventory Script
+ Powershell script that automates the collection of system health metrics and basic inventory by collecting CPU, memory, disk free%, top proccesses and OS version, uptime, and the hostname respectively.    
+ Also Flags common issues like critical service stoppages and low disk space threshold, then sending them to a CSV for monitoring/troubleshooting.
+ Focused on error handling within the script as it will be high-frequency.
